# 🚀 Luna Project - FrontEnd

Bem-vindo ao Luna, um site onde você pode encontrar pessoas para jogar, descobrir novos jogos, jogos em alta no momento e notícias gerais sobre o mundo dos games.
Esta versão é uma MVP (Minimum Viable Product) do que será o projeto futuramente. A MVP foi feita com HTML e CSS básico, com um design exclusivo desenvolvido pela Team Luna - _disponível no [Figma Community](https://www.figma.com/community/file/1559988919515428165)_.

## 📁 Estrutura do Projeto

A estrutura do projeto é organizada para facilitar a manutenção e o desenvolvimento, como exemplificado abaixo:
```
    src/
       ├── assets/                      # Arquivos estáticos
       │   ├── images/                  # Arquivos .png, .jpg, .gif, .webm
       │   ├── icons/                   # Arquivos .svg
       ├── components/                  # Componentes reutilizáveis
       │   ├── GameCard/                # Componente do card do jogo
       │   │   ├── index.html
       │   │   ├── styles.css
       │   ├── SideBar/                 # Componente das barras laterais
       ├── pages/                       # Páginas da aplicação
       │   ├── GamePage/                # Pagina dos Jogos
       │   │   ├── index.html
       │   │   ├── styles.css
       │   ├── NewsPage/                # Página de Notícias
       │   ├── ProfilePage/             # Página do Perfil
       │   ├── FindPeoplePage/          # Página de Encontrar pessoas
       ├── styles/                      # Arquivos de estilização
       │   ├── global.css               # Estilização global
       │   ├── reset.css                # Normaliza estilos e padrões globais
       │   ├── variables.css            # Armazena variáveis de estilo a partir do Design
       ├── helpers/                     # Funções auxiliares (.js)
```
## 🛠️ Ferramentas Utilizadas

- `HTML & CSS`: Para estruturação e estilização básica.
- `Design autêntico`: Desenvolvido pela Team Luna.
